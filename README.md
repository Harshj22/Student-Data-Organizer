# Student-Data-Organizer
About the Project

Student Data Organizer is a beginner-friendly Python project that helps manage student records easily.
It allows users to add, view, search, update, and delete student details — all directly from the terminal.
This project is designed to practice Python fundamentals like lists, dictionaries, tuples, sets, and if-else logic.

Features

1. Add new student records
2. Display all students
3. Search student by ID
4. Update existing record
5. Delete a student record
6. Exit the program safely

Concepts Used

List – to store multiple student dictionaries
Dictionary – to store student data (id, name, age, grade, etc.)
Tuple – to store student ID and Date of Birth together
Set – to store unique subjects
If-Else statements – for menu and options
Loops – for iterating through records
